VoicedGPT

VoicedGPT is a PyQt5 application designed for voice interactions with an AI chatbot.

Key Features:

  • Speech Recognition: Converts voice input to text speech_recognition
  • Natural Language Processing: Uses OpenAI's GPT-3.5 model for AI responses requests
  • Text-to-Speech: Converts AI responses to audible speech pyttsx3
  • Multithreading: Efficiently manages concurrent processes threading
  • Secure Storage: Safely stores sensitive information like API keys dotenv
  • User-Friendly Interface: Includes input/output text areas and a pause button PyQt5
